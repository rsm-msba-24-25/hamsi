Author

Introduction

Dean Karlan at Yale and John List at the University of Chicago conducted a field experiment to test the effectiveness of different fundraising letters. They sent out 50,000 fundraising letters to potential donors, randomly assigning each letter to one of three treatments: a standard letter, a matching grant letter, or a challenge grant letter. They published the results of this experiment in the American Economic Review in 2007. The article and supporting data are available from the AEA website and from Innovations for Poverty Action as part of Harvard‚Äôs Dataverse.

library(dplyr)
library(broom)

# Variables to test
vars <- c("mrm2", "hpa", "years", "freq")

# Create empty lists to hold results
t_test_results <- list()
lm_results <- list()

# Loop through variables
for (var in vars) {
  # --- T-TEST ---
  ttest_result <- t.test(data[[var]] ~ data$treatment, var.equal = TRUE)
  t_test_results[[var]] <- tidy(ttest_result) %>%
    select(statistic, p.value, estimate1, estimate2) %>%
    mutate(variable = var)
  
  # --- REGRESSION ---
  lm_model <- lm(as.formula(paste(var, "~ treatment")), data = data)
  lm_results[[var]] <- tidy(lm_model) %>%
    filter(term == "treatment") %>%
    select(term, estimate, statistic, p.value) %>%
    mutate(variable = var)
}

# Combine into final tables
t_test_df <- bind_rows(t_test_results)
lm_df <- bind_rows(lm_results)

Comparison to Table 1 in Karlan & List (2007)

Table 1 in the Karlan & List (2007) paper provides descriptive statistics for both treatment and control groups before the intervention. Its main purpose is to validate the randomization mechanism by showing that the two groups were similar across observable characteristics.

In this replication, we tested four pre-treatment variables:

  • mrm2 ‚Äì Months since last donation
  • hpa ‚Äì Highest previous contribution
  • years ‚Äì Years since first donation
  • freq ‚Äì Number of previous donations

For each variable, we: - Conducted a t-test to compare means between treatment and control - Ran a linear regression of the form: variable ~ treatment

Commentary on Results

All p-values from both the t-tests and regressions were above 0.05, indicating that none of the differences were statistically significant at the 95% confidence level.

This supports the conclusion that randomization was successful, mirroring the findings in Table 1 of the original paper. Baseline equivalence allows us to attribute post-treatment differences in donation behavior to the effect of the fundraising letter types, not pre-existing differences between groups.

Why Table 1 Matters

Table 1 plays a crucial role because: - It reassures readers that the experimental design was sound. - It strengthens the causal interpretation of later results. - It provides replicability and transparency via descriptive stats for key variables.
